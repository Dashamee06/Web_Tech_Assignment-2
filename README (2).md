# Online Registration Form Application

A web-based registration form built with HTML, CSS, JavaScript, jQuery, and PHP.

## Features
- Responsive registration form with validation
- Real-time form submission using AJAX
- Styled success display with formatted data
- Server-side validation and sanitization

## Local Testing

1. Install a local PHP server (XAMPP, WAMP, or MAMP)
2. Place files in the web server directory (e.g., `htdocs` for XAMPP)
3. Start Apache server
4. Access via `http://localhost/[folder-name]/`

## Files Structure
- `index.html` - Main form page
- `style.css` - Styling and layout
- `script.js` - jQuery form handling
- `process.php` - Backend processing
- `README.md` - Documentation
